# zomato-data-analysis-dashboard
This repository contains an interactive Power BI dashboard built on Zomato restaurant data. The dashboard analyzes restaurant distribution, ratings, cuisines, cost ranges, and city-wise trends. It demonstrates data cleaning, modeling, DAX calculations, and effective visualization techniques for data analytics and business intelligence applications.

# Zomato Insights Dashboard (Power BI)

## Overview
This project features an interactive Power BI dashboard created using Zomato restaurant data. The dashboard provides insights into restaurant performance, customer ratings, cuisine distribution, and pricing trends across different cities.

## Objectives
- Analyze restaurant data from Zomato
- Understand city-wise restaurant distribution
- Evaluate ratings and average cost trends
- Identify popular cuisines
- Practice Power BI data modeling and DAX

## Key Insights
- Restaurant count by city
- Distribution of ratings
- Popular cuisines analysis
- Price range and average cost comparison
- Online delivery and table booking trends

## Tools & Technologies
- Power BI
- Power Query (Data Transformation)
- DAX (Measures & Calculations)
- Data Visualization & Dashboard Design
